Instructions

  • Select a square by clicking on it or using the arrow keys
  • Type a number from 1 to 9 in the selected square
  • Use the backspace key to delete the current value in the square
  • When the puzzle is ready, use the buttons below the puzzle grid to solve
  • You can also input a pre-made puzzle using the corresponding button
